# EyePay

![EyePay Banner](docs/banner/eyepay-banner.png)

## AI-Powered Global Remittance & Digital Payments Platform

EyePay is a modern fintech platform designed for secure global remittances, wallet management, fraud detection, and intelligent financial insights.

### Live Demo

**Frontend:** https://eye-pay-eyepay.vercel.app

**Backend:** https://eyepay-1.onrender.com

---

## Features

- JWT Authentication
- Wallet Management
- Global Money Transfers
- Recipient Management
- Fraud Detection
- Analytics Dashboard
- AI Intelligence Dashboard
- PostgreSQL + Drizzle ORM
- Cloud Deployment
